# 🎯 Jogo da Forca

Um Jogo da Força moderno e interativo, desenvolvido em HTML5, CSS3 e JavaScript puro, com visual animado, sistema de pontuação, temas visuais por categoria e suporte a palavras e frases com espaços.

## 🧠 Sobre o Projeto

Este projeto é uma versão aprimorada do clássico Jogo da Força, trazendo uma experiência visual mais rica e dinâmica, além de uma base de palavras e frases amplas e variadas.

O jogo escolhe aleatoriamente uma categoria, aplica um tema visual exclusivo, e desafia o jogador a adivinhar palavras ou frases completas, incluindo nomes compostos como:

"São Paulo"

"Rio de Janeiro"

"Desenvolvimento Web"

"O Poderoso Chefão"

## 🎮 Como Jogar

Ao iniciar, uma palavra ou frase é classificada automaticamente

A categoria aparece no topo da tela

Clique nas letras para tentar adivinhar

Cada erro reduz uma vida ❤️

Cada acerto soma pontos ⭐

Você vence ao completar a palavra/frase antes de acabar as vidas

Clique em 🔄 Novo Jogo para jogar novamente
